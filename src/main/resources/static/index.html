Rollback Banking API Console

Spring Boot REST API를 브라우저에서 호출해 보는 단일 페이지(HTML5 + CSS + JS). 기본 포트는 8080 입니다.

고객 생성

POST /api/banking/customers
샘플 데이터: 고객 ID 1~3, 고객번호 CUST001~003 (schema.sql)

PUT / POST

고객 목록

계좌 조회

GET /api/banking/accounts
샘플 데이터: 계좌 ID 1~4, 계좌번호 ACC001~004 (schema.sql)

계좌 상태 변경

POST /api/banking/accounts/{id}/freeze
